Weighted Kappa

This procedure provides an estimate of weighted kappa using linear or quadratic weighting, along with asymptotic standard error, Z statistic, significance or p value under the null hypothesis of chance agreement and confidence interval for kappa. The Z statistic and p value are computed using an asymptotic standard error (ASE) that assumes the null hypothesis (sometimes denoted ASE0). The ASE shown does not assume the null hypothesis (sometimes denoted ASE1). The confidence intervals are computed using this ASE1 value. Formulas are taken from Fleiss, Levin & Paik (2003).

►To run this procedure, from the menus choose:

 Analyze
  Scale
   Weighted Kappa...

►Specify two Numeric Rating Variables. These cannot be date or time variables. The possible rating categories must be consective positive integers beginning with 1.

Optionally, you can:

• Specify the power for weighting: 1 (linear) or 2 (quadratic). The default is 1.

• Specify confidence interval coverage levels from 50% to 99.999%. The default is 95%.

Additional Features

Run the command

STATS WEIGHTED KAPPA /HELP.

to display the entire syntax description.

Data Considerations

Split files are ignored. Case filtering is honored. Cases missing on any specified variable(s) are not used. Weights are accepted but are rounded to integers.

Reference

Fleiss, J. L., Levin, B., & Paik, M. C. (2003). Statistical methods for rates and proportions (3rd ed.). Hoboken, NJ: John Wiley & Sons.
