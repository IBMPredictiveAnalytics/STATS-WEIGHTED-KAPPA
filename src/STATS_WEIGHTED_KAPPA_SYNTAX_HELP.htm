STATS WEIGHTED KAPPA Extension Command

Calculate weighted kappa statistic and associated quantities. This procedure provides an estimate of weighted kappa using linear or quadratic weighting, along with asymptotic standard error, Z statistic, significance or p value under the null hypothesis of chance agreement and confidence interval for kappa. The Z statistic and p value are computed using an asymptotic standard error (ASE) that assumes the null hypothesis (sometimes denoted ASE0). The ASE shown does not assume the null hypothesis (sometimes denoted ASE1). The confidence intervals are computed using this ASE1 value. Formulas are taken from Fleiss, Levin & Paik (2003).

Note: This extension command does not support split files. All complete cases in the active dataset are used. Cases missing on any specified variable(s) are not used. Weights are accepted but are rounded to integers.

STATS WEIGHTED KAPPA VARIABLES=varlist*
/OPTIONS WTTYPE=number (1)** CILEVEL=percentage (95)**

/HELP

* Required
** Default

/HELP displays this help and does nothing else.



STATS WEIGHTED KAPPA VARIABLES=rating1 rating2
/OPTIONS WTTYPE=1 CILEVEL=95.

VARIABLES is the list of rating variables. Exactly two variables must be specified (the "TO" and "ALL" conventions are allowed). These cannot be date or time variables. The possible rating categories must be consecutive positive integers beginning with 1.

WTTYPE is the weighting power. It must be 1 (for linear weighting) or 2 (for quadratic weighting). It defaults to 1.

CILEVEL is the confidence level for confidence intervals expressed as a percentage. It cannot be less than 50 or greater than 99.999. It defaults to 95.


Reference

Fleiss, J. L., Levin, B., & Paik, M. C. (2003). Statistical methods for rates and proportions (3rd ed.). Hoboken, NJ: John Wiley & Sons.
